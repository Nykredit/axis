# Stack Component

This component allows you to layout children with specified spacing both vertically and horizontally.

<Canvas of={Stories.Example} />

## Description

The stack component is used to layout children with a specified spacing in both directions. To control both vertical and horizontal spacing, use multiple nested stack's .

## Usage

`<nk-stack></nk-stack>`

## Usage notes

### Do:

- Use this to define spacing between components

### Do not:

## @API

### Properties

- variant (`spacing-none`, `spacing-min`,`spacing-tiny`,`spacing-xsmall`,`spacing-small`,`spacing-medium`,`spacing-xlarge`,`spacing-max`)
- orientation (horizontal | vertical)
- alignment (none | start | center | end)
- wrap (wrap | nowrap)
