# Single select card item

Demo component showing how to use single select card item

<Canvas of={Stories.Default} />

## Description

This component was created in order to be able to select a card with both image and content.

## Usage

`<nk-card-select-single></nk-card-select-single>`

## Child components

This component is a componsition of the following

- card-select

## @API

### Properties

- variant (`string`)
- imageurl (`string`)
- imagealttext (`string`)
- actiontitle (`string`)
- actionlabel (`string`)
- actionvalue (`boolean`)
- cardid (`string`)

### Event handlers

- Add a onchange handler on `card-select`. The handler returns the target and actionvalue at it's current state.
