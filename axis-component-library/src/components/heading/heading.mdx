# Heading Component

Heading component for rendering a piece of text as a heading

<Canvas of={Stories.Example} />

## Description

The heading allows you to render a heading or sub heading, depending on the specified variant of the specified h\* level.

## Usage

`<nk-heading></nk-heading>`

### Do:

- Use the component to render your heading tree

### Do not:

- Use as a purely visual element or other text element which is not strictly a heading

## @API

### Properties

- text (`string`)
- variant (`huge`, `xlarge`, `large`, `medium`, `small`, `xsmall`, `tiny`)
- level (`1`,`2`,`3`,`4`,`5`,`6`)
