# IconButton component

Button with an icon instead of label

<Canvas of={Stories.Add} />

## Description

The IconButton is a minimal button, where instead of a visible label, the button contains an icon. The IconButton is typically used to add or remove items in lists, where labeled buttons may look to distracting.

## Usage

`<nk-button-icon></nk-button-icon>`

### Do:

- Use the icon button where the context is clear

### Do not:

- Use icon buttons if the label is required to understand the function of the button.
- Use the same IconButton for multiple different unrelated functions

## Child components

This component uses the **Icon** component

## @API

### Properties

- icon (string) - the name of the icon
- variant (string) - The style of the button
  - `filled/primary`
  - `neutral/secondary`
  - `ghost/tertiary`
  - `outline`
