# Button Component

Primary component for executing an action that does not result in page navigation

<Canvas of={Stories.Example} />

## Description

The button element is the go-to component for executing an action when clicking. Such actions could be closing a modal, sending a form or adding/removing an element from a list.
The button has 3 variants, primary, secondary and tertiary with diminishing visual weight.

## Usage

`<nk-button></nk-button>`

## @API

### Properties

- label (`string`)
- leadingicon (`string`)
- variant (`filled`,`neutral`, `outline`, `ghost`)
- trailingicon (`string`)
- disabled (`boolean`)

### Event handlers

- click (CustomEvent) - Event fired when the button is clicked
