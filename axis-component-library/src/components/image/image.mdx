# Image component

## Description

This component was created in order to be able to add an image with different variants for cardSelect.

## Usage

`<nk-image></nk-image>`

## @API

### Properties

- url (`string`)
- alt (`string`)
- variant (`cover`, `contain`, `fill`, `none`)
